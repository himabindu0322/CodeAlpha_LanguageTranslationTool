# 🌐 CodeAlpha Language Translation Tool

A powerful and user-friendly Language Translation Tool developed using **Python, Streamlit, and Google Translate API**. This application allows users to translate text between multiple languages, listen to translations using Text-to-Speech, and download translated audio files.

## 🚀 Features

* 🌍 Translate text into multiple languages
* 🔄 Real-time translation using Google Translate
* 🔊 Text-to-Speech (TTS) support
* 🎵 Play translated audio directly in the app
* ⬇ Download translated audio as MP3
* 📋 Copy translated text to clipboard
* 📜 Translation History stored during the session
* 🗑 Clear Translation History
* 🎨 Professional dark-themed UI with custom CSS
* ⚡ Fast and easy-to-use interface

## 🛠️ Technologies Used

* Python
* Streamlit
* Deep Translator
* Google Text-to-Speech (gTTS)
* Pyperclip

## 📦 Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 🌐 Supported Languages

The application currently supports:

* English
* Hindi
* Telugu
* Tamil
* French
* German
* Spanish
* Chinese
* Japanese
* Arabic

More languages can be added easily.

## 🧠 Working Principle

1. User enters text.
2. Select source and target languages.
3. The application sends the text to Google Translate API.
4. The translated text is displayed instantly.
5. Users can:

   * Copy the translation
   * Listen to the translation
   * Download translation audio
6. Translation history is saved during the session.
